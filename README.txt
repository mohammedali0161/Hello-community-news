README - Community News Website (Simple Version)

How to add a new article:
1. Go to the 'articles' folder.
2. Create a new text file (e.g., article2.txt).
3. Use this format:
   Title: Your Article Title
   Date: DD Month YYYY
   Image: Image URL (online image recommended)
   Content: Write your article content here.
4. Save the file and refresh index.html in your browser.
5. To display it automatically, add the filename to the 'articleFiles' array in index.html:
   const articleFiles = ['articles/article1.txt', 'articles/article2.txt'];

Viewing the website:
- Open index.html in any browser (iPhone, PC, or Mac) to see your articles.

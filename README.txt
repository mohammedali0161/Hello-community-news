README - Community News Website

How to add a new article:
1. Open index.html in a text editor.
2. Scroll to the JavaScript section at the bottom (starts with "const articles = [").
3. Add a new article like this:
   {
     title: "Your Article Title",
     date: "DD Month YYYY",
     image: "Image URL",
     content: "Write your article content here."
   },
4. Save the file and refresh your browser to see the new article.

How to publish online:
1. Purchase a domain and hosting that supports HTML.
2. Upload all files from this folder to the hosting server.
3. Open your domain in a browser to view your live website.
